

# OpenQASM Language Support

This extension provides VSCode, Eclipse, and Theia IDE's support for the OpenQASM language

![in-line and block commenting](./images/commenting.png)

## Features

* Syntax-Highlighting
* Multi-line Comments
* Advanced Loop Styling
